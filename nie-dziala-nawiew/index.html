Nie działa nawiew — znajdź swój model

Bezpiecznik, rezystor, silnik dmuchawy. Krok po kroku dla każdego popularnego modelu.

Szybka diagnostyka — 3 pytania

  • Działa tylko bieg 4? → Uszkodzony rezystor dmuchawy (najczęstszy przypadek)
  • Nie działa żaden bieg? → Sprawdź bezpiecznik w skrzynce pod deską
  • Bezpiecznik OK, napięcie jest, ale nie kręci? → Wymiana silnika dmuchawy
